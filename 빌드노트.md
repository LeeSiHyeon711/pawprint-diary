# 빌드노트 — PawprintDiary

## 프로젝트 정보
- 스택: Next.js 14 App Router + React 18 + TypeScript + Tailwind CSS
- 위치: `projects/PawprintDiary/05-개발/` (이 폴더가 Next.js 프로젝트 루트)

---

## 실행 방법

### 1. 의존성 설치
```bash
cd projects/PawprintDiary/05-개발
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```
브라우저에서 `http://localhost:3000` 접속

### 3. 프로덕션 빌드 검증
```bash
npm run build
```
타입/빌드 에러 0으로 통과해야 FEAT 완료 조건 충족 (설계서 7장 전역 규칙)

---

## 환경 변수 설정

`.env.example`을 복사해 `.env.local`로 만들고 API 키를 입력합니다:
```bash
cp .env.example .env.local
# .env.local 에서 ANTHROPIC_API_KEY= 에 실제 키 입력
```

**ANTHROPIC_API_KEY가 없어도 Mock 모드로 전체 UI 흐름이 동작합니다** (실제 Mock 구현은 FEAT-05).  
키를 넣으면 실제 `claude-opus-4-8` 모델을 호출합니다.

## 이슈별 작업 기록

### FEAT-01 — 프로젝트 셋업 + 디자인 시스템 (이슈 #1)
**추가 문서 읽기 이유 기록**: FEAT 문서가 "설계서 4-3 컬러표 그대로"를 명시했으나 hex 값이 FEAT 문서에 없어 설계서를 1회 추가 확독.

**구현 완료 파일 목록**:
- `package.json` — 의존성 정의 (next@14, react@18, tailwindcss, idb, @anthropic-ai/sdk, date-fns)
- `next.config.mjs` — Next.js 기본 설정
- `tsconfig.json` — TypeScript + path alias(@/*)
- `tailwind.config.ts` — 디자인 토큰 매핑 (colors, borderRadius, boxShadow, fontFamily)
- `postcss.config.mjs` — PostCSS (Tailwind + Autoprefixer)
- `.env.example` — ANTHROPIC_API_KEY(빈 값), ANTHROPIC_MODEL=claude-opus-4-8
- `app/globals.css` — CSS 변수 토큰 + Tailwind 베이스 + Pretendard CDN
- `app/layout.tsx` — 루트 레이아웃 (PetProvider 임시 + BottomNav)
- `app/page.tsx` — 임시 환영 페이지 (/today 링크 포함)
- `components/ui/Button.tsx` — variant(primary/soft/ghost), full prop
- `components/ui/Card.tsx` — radius 16, surface 배경, 옅은 그림자
- `components/ui/Tag.tsx` — pill 칩 (선택형/표시형)
- `components/ui/Field.tsx` — label + children 묶음
- `components/ui/EmptyState.tsx` — 빈 상태 안내 (🐾 기본 아이콘)
- `components/ui/Spinner.tsx` — 로딩 스피너
- `components/ui/SectionTitle.tsx` — 섹션 제목
- `components/BottomNav.tsx` — 하단 4탭 (오늘/기록/질문/프로필, 활성 표시)
- `components/AppHeader.tsx` — 상단 헤더 (타이틀 + 옵션 children)
- `lib/petContext.tsx` — PetProvider/usePet 임시 골격 (FEAT-03에서 교체 예정)

**빌드 결과**: Bash 차단으로 직접 실행 불가. 공방장이 아래 명령으로 검증 필요:
```bash
cd projects/PawprintDiary/05-개발 && npm install && npm run build
```
